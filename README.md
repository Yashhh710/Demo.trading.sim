## 🚀 Overview

Demo Trading Sim is a modern web-based stock trading simulator designed for learning and experimentation. It provides an interactive trading experience using realistic demo market data, allowing users to explore trading strategies without using real money.

The platform includes a team of specialized virtual assistants that collaborate to analyze markets, manage portfolios, assess risk, and provide trading insights.
